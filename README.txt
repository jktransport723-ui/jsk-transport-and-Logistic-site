JSK Transport and Logistics - Vite React site

This package is ready to deploy. Steps to run locally:

1. Install dependencies:
   cd jsk-vite-site
   npm install

2. Run dev server:
   npm run dev

3. Build for production:
   npm run build

4. The build output will be in the /dist folder. Upload the contents of /dist to Netlify (drag-and-drop) or connect the repository to Netlify/Vercel.

Notes:
- Logo is included at src/assets/business logo.jpg
- PDFs are in public/downloads/
- Vite + React are configured in package.json
